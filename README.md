<h2>📌 Sobre o Projeto</h2>

<p>
  A <strong>Saint Peter Technology</strong> é uma solução de monitoramento que protege seus eletrocardiógrafos, oferecendo dados precisos e de valor em seu negócio, para você sempre tomar a melhor decisão exatamente onde importa.
</p>

## 📦 Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/Saint-Peter-Tech/SaintPeter-web
```

2. Entre no diretório:

```bash
cd web
```

3. Instale os pacotes:

```bash
npm i
```

4. Execute o projeto:

```bash
npm start
```
